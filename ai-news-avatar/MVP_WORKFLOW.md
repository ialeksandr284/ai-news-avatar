# MVP Workflow

## Stage 1. Fetch
Collect candidate stories from trusted AI sources.

## Stage 2. Select
Choose one news item that:
- is recent
- is about a model, engine, inference stack, API, or major release
- can be explained in one short

## Stage 3. Summarize
Generate a factual Russian brief:
- что произошло
- кто выпустил
- что нового
- почему это важно
- на что обратить внимание

## Stage 4. Write Script
Generate a Russian script for a 30-second avatar short.

Structure:
1. Hook
2. Main news
3. Why it matters
4. Clean close

## Stage 5. Generate Assets
- title
- description
- hashtags
- thumbnail brief

## Stage 6. Voice + Avatar
- TTS from final Russian script
- avatar render in vertical format

## Stage 7. Review
Current mode:
- present output in workspace

Future mode:
- send to Telegram with preview link

## Stage 8. Publish
Upload approved output to YouTube Shorts with metadata.
