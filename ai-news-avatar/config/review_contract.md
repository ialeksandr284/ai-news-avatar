# Review Contract

## Current Review Mode
Manual review inside this workspace.

What must be shown before publish:
- selected headline
- source links
- Russian script
- draft title
- draft description
- video preview link or file

## Telegram Review Mode
Later, the pipeline should send a Telegram message with:
- headline
- 1-2 line summary
- source links
- preview link
- approve / reject action

## Approval Rule
Do not publish automatically without explicit approval.
